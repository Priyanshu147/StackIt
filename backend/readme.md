# Backend


# Create a new file named .env

```
# JWT secrets for applicant authentication
APPLICANT_ACCESS_TOKEN_SECRET=
APPLICANT_REFRESH_TOKEN_SECRET=

# JWT secrets for user authentication
USER_ACCESS_TOKEN_SECRET=
USER_REFRESH_TOKEN_SECRET=

# Application port
PORT=8000

# MySQL database configuration
MYSQL_HOST=your-host-ip
MYSQL_USER=your-db-username
MYSQL_PASSWORD=your-db-password
MYSQL_DATABASE=your-db-name

# API Key to validate incoming requests (should match frontend key)
BACKEND_API_KEY=
```
